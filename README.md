# Book Tracker

**Автор:** Иван Иванов

## Описание программы

Приложение для учёта прочитанных книг с фильтрацией и сохранением данных в JSON.

## Инструкция по запуску

1. Установите Python 3.
2. Установите зависимости: `pip install -r requirements.txt` (если есть).
3. Запустите: `python book_tracker.py`

## Примеры использования

- Добавьте книгу через форму.
- Используйте фильтры по жанру и количеству страниц.
- Сохраняйте и загружайте данные.
